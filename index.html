📁 轻量级文件传输工具

Python Socket 服务端 + PyQt5 客户端 · 多客户端互传 · 实时中继 · 无需安装额外服务

🖥️ 服务端 server.py

  • 多线程 每个客户端独立线程，互不阻塞
  • 文件中继 流式转发，服务端不落盘，内存友好
  • 用户管理 注册、在线列表、下线清理
  • 广播 用户列表变动自动推送所有客户端
  • 聊天 支持私聊和广播消息转发
  • 纯标准库 socket + threading，零依赖

💻 客户端 client.py

  • PyQt5 GUI 现代 Fusion 风格界面，跨平台
  • 进度条 大文件发送实时显示进度
  • 自动接收 来文件时后台自动保存到指定目录
  • 在线列表 实时刷新，双击切换私聊对象
  • 不阻塞 UI 网络收发在独立 QThread 中运行
  • 可配置 服务器地址/端口/接收目录均可自定义

🏗️ 架构示意

📡 通信协议消息类型

type方向说明
registerClient→Server注册用户名，首条必须发送
register_okServer→Client注册成功确认
get_usersClient→Server请求在线用户列表
user_listServer→Client推送在线用户列表（上下线自动广播）
file_requestClient→Server发起文件传输，header 后紧跟二进制 payload
incoming_fileServer→Client通知接收方有文件，紧接着跟二进制数据
file_sent_okServer→Client通知发送方中继完成
file_received_okServer→Client通知双方传输成功
chat双向聊天消息，to 字段为空则广播
errorServer→Client错误消息

🚀 快速开始

1

安装依赖（服务端无需安装）

pip install PyQt5
2

启动服务端

python server.py # 默认 0.0.0.0:9999 python server.py --host 0.0.0.0 --port 8888 # 自定义端口
3

启动客户端（可多开）

python client.py
4

在 GUI 中操作

填写服务器地址 + 端口 + 用户名 → 连接 选中在线用户 → 点击「发送文件」→ 选择文件 → 自动传输 接收方自动收到文件并保存到指定目录（默认桌面）
